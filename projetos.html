Páginas web

Projeto HTML - Saúde Paraná

Website informativo desenvolvido com HTML e CSS como projeto da disciplina Desenvolvimento Front-End da graduação em Análise e Desenvolvimento de Sistemas, que apresenta algumas informações, como sintomas comuns e meios de prevenção atestados pelo Ministério da Saúde, sobre as doenças endêmicas de maior incidência no estado do Paraná.

Obs: Nosso objetivo não é incentivar o auto-diagnóstico, em caso de sintomas procure ajuda médica.

Projeto React - Catálogo de Jogos

Website desenvolvido com React, que exibe ofertas de jogos eletrônicos da plataforma Steam em um grid de vizualização paginado. Os dados são obtidos da API CheapShark.

Projeto HMTL - Universo C

Página web desenvolvida utilizando HTML e CSS, como projeto da disciplina Certificadora de Competência Comum da graduação em Análise e Desenvolvimento de Sistemas. Consiste em uma plataforma com vídeoaulas e apostilas em PDF de um curso introdutório de Lógica de Programação com Linguagem C. Aplicativos móveis

Projeto Flutter - Coversor de Moedas

Aplicativo para Android, desenvolvido com Flutter, que realiza conversões entre moedas (Real, Dólar, Euro e Bitcoin) utilizando uma API (AwesomeAPI) para obter os valores atualizados de cada moeda.

Projeto Flutter - Buscador de GIFs

Aplicativo para Android, desenvolvido com Flutter, que permite ao usuário visualizar, pesquisar (por meio de filtros inseridos na requisição) e compartilhar GIFs obtidos da API do site GIPHY.

Projeto Flutter - Agenda de Contatos

Aplicativo para Android, desenvolvido com Flutter, no qual o usuário pode cadastrar informações de um contato (nome, e-mail, telefone e uma foto), bem como editá-las e removê-las.
